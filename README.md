# Masterblog - Simple Flask Web Application

A lightweight, local blogging web application built with **Python** and **Flask**. This project was developed as a practical hands-on exercise during a software development training course to master the fundamentals of web routing, backend logic, template rendering, and basic CRUD operations without using an external database.

## 🚀 Features

- **Read (Display Posts):** Dynamically loads and displays blog posts on the main page from a local JSON file.
- **Create (Add Posts):** A dedicated form to publish new articles with automatically generated IDs.
- **Update (Edit Posts):** Allows users to modify existing posts using pre-filled forms for better user experience.
- **Delete Posts:** Includes a JavaScript confirmation popup to securely delete articles.
- **Interactive "Like" Button:** A dynamic feature that increments a post's like counter instantly.

## 🛠️ Tech Stack

- **Backend:** Python 3, Flask Framework
- **Frontend:** HTML5 (Jinja2 Templates), CSS3 (Custom styling)
- **Data Storage:** Local JSON file (`blog_data.json`)
